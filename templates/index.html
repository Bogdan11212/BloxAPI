Getting Started

Quick Start Guide

Here's a simple example of how to use the API to get user information:

import requests # Get user information response = requests.get('http://localhost:5000/api/users/1234567') user_data = response.json() print(user_data)

Search for users by username:

import requests # Search for users response = requests.get('http://localhost:5000/api/users/search?keyword=builderman&limit=5') search_results = response.json() print(search_results)

Get game details:

import requests # Get game details response = requests.get('http://localhost:5000/api/games/1234567890/details') game_details = response.json() print(game_details)
