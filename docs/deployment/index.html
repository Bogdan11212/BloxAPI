General Deployment Requirements

  • Python 3.7+ - BloxAPI requires Python 3.7 or higher
  • PostgreSQL database - For storing user data and analytics
  • Environment Variables - API keys and configuration settings

Environment Variables

The following environment variables should be configured regardless of deployment platform:

  • SECRET_KEY - A secret key for secure session handling
  • DATABASE_URL - PostgreSQL connection string
  • ROBLOX_API_KEY - Your Roblox API key (optional)
  • REDIS_URL - Redis connection string (optional, for caching)
